# birthday

Validate birthday, parse the age and constellation of valid birthday.

## Install

```
composer require dakalab/birthday
```

## Usage

```
$birthday = new Birthday('1988-01-01', 'cn');
$age = $birthday->getAge();
$constellation = $birthday->getConstellation();
```
